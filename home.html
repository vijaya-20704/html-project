HTML INTERVIEW QUESTIONS

1)WHAT IS HTML?

HTML (HyperText Markup Language) is the standard language used to create and design web pages.

2)WHAT IS THE PURPOSE OF DOCTYPE IN HTML?

The in HTML tells the web browser which version of HTML the page is written in. It helps the browser display the page correctly.

3)WHAT IS THE DIFFERENCE BETWEEN INLINE AND BLOCK-LEVEL ELEMENTS IN HTML?

Block-level elements take up the full width of the page, starting on a new line. Inline elements only take up as much width as needed and stay in the same line.

4)WHAT IS THE USE OF THE ANCHOR TAG?

The anchor tag is used to create links in a webpage. It lets you connect to another page, website, file, or section on the same page. When you click it, it takes you to the linked location.

5)WHAT ARE HTML INPUT TYPES?

HTML input types define the kind of data a user can enter in a form. They are used with the input tag to get different types of input like text, numbers, dates, files, etc.
